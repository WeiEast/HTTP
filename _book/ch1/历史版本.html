2.历史版本

HTTP协议至今公布的版本有3个：

  • HTTP 0.9，只有基本的文本 GET 功能。
  • HTTP 1.0，完善的请求/响应模型，并将协议补充完整。
  • HTTP 1.1，1999 年 6 月 HTTP 1.1 RFC 2616发布，在 1.0 基础上进行更新，增加了如 长久连接 keep-alive 与 chunked 等功能。
